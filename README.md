## Running Locally

1. Install dependencies:

```sh
pnpm install
```

2. Copy the `.env.example` file to `.env`:

```sh
cp .env.example .env
```

3. Start the dev server:

```sh
pnpm dev
```

## Building for Production

1. Build the app:

```sh
pnpm build
```

2. Start the production server:

```sh
pnpm start
```
